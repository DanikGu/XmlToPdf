Put the index-term in the place where an index is generated in an actual document, and then place <fo:index-page-citation-list>. <fo:index-page-citation-list> enumerates <fo:index-key-reference> as a child element of it. <fo:index-key-reference> shows the page number to which the index term belongs. By using the ‘ref-index-key’ property, a key value of an index term is specified. Also, whether the reference page is shown as a hyperlink can be specified with the ‘page-number-treatment’ property.
